## Data Collection

GitHub PR Visual Editor does **not** collect, store, or transmit any user data. The extension runs entirely locally in your browser.

## Permissions

- **activeTab**: Used to inject the content script on GitHub pages to enable editable preview functionality.
- **Host permission (github.com)**: Required to run on GitHub Pull Request pages and modify the page DOM.

## Third-Party Services

This extension does not communicate with any external servers, APIs, or third-party services. No analytics, tracking, or telemetry of any kind is used.

## Changes

If this policy changes, it will be updated in this repository.
